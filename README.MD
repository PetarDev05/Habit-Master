# Habit Master

Habit Master is a web application designed for weekly habit tracking, with a strong focus on clarity, consistency, and well-structured data.

The application allows users to:
- create weekly tracking periods with a defined start and end date
- add multiple habits within a single week
- track daily check-ins for each habit
- mark habits as completed, missed, or skipped
- review historical data and analyze long-term progress through completed weeks

This project is being developed as a personal, long-term software project with the goal of building a stable, scalable, and professionally structured full-stack application.

---

## Technology Overview

- **Backend:** Node.js, Express, MongoDB (Mongoose)
- **Authentication:** JWT-based authentication with access and refresh tokens
- **Frontend:** Planned (not implemented yet)
- **Architecture:** Modular, layered backend architecture with clear separation of concerns
